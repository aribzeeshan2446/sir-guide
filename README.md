# SIR Guide — Telangana Electoral Roll Revision 2026 🗳️

A clean, mobile-first public awareness website built to guide Telangana citizens through the 2026 Special Intensive Revision (SIR) of electoral rolls.

**🌍 Live Website:** [https://sir-guide.vercel.app](https://sir-guide.vercel.app)

---

## 🎯 The Purpose

In 2025/2026, the Election Commission of India (ECI) initiated a massive door-to-door enumeration drive in Telangana to map current voters back to the **2002 electoral rolls**. Citizens who cannot be mapped back to these records may be required to prove their eligibility. 

This project aims to provide a clear, step-by-step roadmap for citizens to:
1. Search the 2002 electoral roll to find their family's records.
2. Determine their next action steps (visiting GHMC Help Desks).
3. Gather the required documentation as a backup plan.

## ✨ Features

- **Professional Editorial Design:** A serious, civic-focused aesthetic utilizing clear typography and semantic status colors.
- **Multi-language Support (i18n):** Instantly switch between **English**, **Telugu (తె)**, Urdu (اُرْدُو) and **Hindi (हि)** without reloading the page. Uses localized Google Noto Sans fonts for crisp rendering.
- **Fully Responsive:** Mobile-first layout with a custom hamburger menu and responsive spacing.
- **Zero Dependencies:** Built using vanilla HTML, CSS, and JavaScript. Extremely fast and lightweight.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, and modern typography styling.
- **Vanilla JavaScript:** Sticky navigation, mobile menu toggles, scroll-reveal animations, and instant i18n language switching.

## 🚀 Development & Deployment

To run this project locally, simply clone the repository and open `index.html` in your browser, or run a local server:

```bash
git clone https://github.com/aribzeeshan2446/sir-guide.git
cd sir-guide
npx serve .
```

This project is deployed automatically via **Vercel**. Every push to the `main` branch triggers a new deployment.
